# Need For Speed Most Wanted Traffic Racer Edition

![nfsmw](https://cdna.artstation.com/p/assets/images/images/039/001/058/large/-nfs-mw-3.jpg?1624651166)

### Gameplay
  Move using:
- `Up arrow ↑` or `W`
- `Left arrow ←` or `A`
- `Down arrow ↓` or `S`
- `Right arrow →` or `D`
