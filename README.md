Need For Speed Most Wanted Traffic Racer Edition

![https://www.dsogaming.com/wp-content/uploads/2020/04/Need-for-Speed-Most-Wanted-2005.jpg]

Gameplay
  Move using:
- `Up arrow ↑` or `W`
- `Left arrow ←` or `A`
- `Down arrow ↓` or `S`
- `Right arrow →` or `D`
